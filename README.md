# 🎙️ Friday AI

**Friday AI** is a browser-based AI Voice Assistant built using **React**, the **Web Speech API**, and the **Wikipedia REST API**. It can understand voice commands, respond using speech synthesis, search the web, open popular websites, play YouTube videos, and answer general knowledge questions.

## ✨ Features

- 🎤 Voice Recognition
- 🔊 AI Voice Responses (Text-to-Speech)
- 🌐 Open Popular Websites
- 🔍 Google Search
- ▶️ YouTube Search
- 🎵 Play Videos on YouTube
- 📖 Wikipedia Information Search
- 🕒 Current Time
- 📅 Current Date
- 👋 Greeting Detection
- 🛑 Start / Stop Listening
- ✅ Automatically Stops After One Command
- 🚫 Ignores Empty Voice Commands
- 🚫 Prevents Self-Triggering During Speech Output

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/friday-ai.git
```

Navigate to the project

```bash
cd friday-ai
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
